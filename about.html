# 💍 Wedding Event Management Website (Frontend) ## 📌 Project Overview The **Wedding Event Management Website** is a frontend web application designed to simplify the wedding planning process. It provides a clean and user-friendly interface where users can explore wedding services, manage plans, and organize different aspects of their wedding. This project focuses on **UI/UX design and client-side functionality** using modern web technologies. --- ## 🎯 Objectives * Design an attractive and responsive wedding website * Provide an easy-to-use interface for planning events * Showcase wedding services like venues, vendors, and guest management * Enhance user experience with smooth navigation and visuals --- ## ✨ Features ### 👰 User Interface * 🏠 Home page with wedding theme design * 📋 Services section (venue, catering, photography, etc.) * 📞 Contact page * 📸 Image galleries for inspiration ### 🧩 Functional Features * 📅 Wedding planning sections (UI-based) * 👥 Guest list layout (static/demo) * 🎯 Interactive buttons and navigation * 📱 Fully responsive design --- ## 🛠️ Technologies Used | Technology | Purpose | | ---------- | ---------------------------------- | | HTML | Structure of the website | | CSS | Styling and layout | | JavaScript | Interactivity and dynamic behavior | --- ## 📂 Project Structure ```bash wedding-event-management-website/ │── index.html │── css/ │ └── styles.css │── js/ │ └── script.js │── images/ │── pages/ ``` --- ## 🖼️ Website Preview *Add screenshots of your website here* --- ## 🚀 How to Run 1.
